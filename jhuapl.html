June 2017 - August 2017

In my sophore year of highschool I intered at the Johns Hopkins Applied Physics Lab(JHUAPL). My first tast assigned was to model the motion of a rocket using Python. This first project was a teaching lesson used to teach me about the forces acting on a rocket, the different types of coordinate systems (body fixed, wind axes), and how ODE functions work.


After I finished that I got to work with a Hypersonic Panel Code. The Panel Code projects objects into hypersonic flow (flow speed above Mach 5) and calculates the aerodynamic forces acting on the body. (To learn more about how the panel code works, click here). The Panel Code only excepts a list of points as an input. So a user would have to manually define the point generation of the body in MATLAB to use the Panel Code. I created an automesh system that would allow STL(CAD) files to be used as an input so the user didn't have to manually define the body. My system broke the an STL file into triangles, and generated points within each triangle to get an outline of the body.


The last project I worked on my sophmore year was cadding a trident missile. I cadded all three stages of the rocket. One the surfaces where each stage connected, I CADed in holes so magnets could be superglued in there. This way, the stages could be pulled apart or magnetically connected. After I finished CADding the rocket, my mentor and I 3D printed it. It now sits in his office, and he takes it to corporate events.


June 2018 - August 2018

I had so much fun that I returned back to my lab the next year. I created a controller to determine the stability of an object in the Panel Code at different angles of attack and beta angles.
I derived my A and B matrices by taking the partial derivates of my state(x vars) and input(u vars) variables. I then ran the panel code at different angles of attack, beta angles, and fin deflection angles to get numberical values for my A and B matrices. I used this to determine stability of the object.

This project was a big learning experience for me. Before that summer, I had no clue what a partial derivate was, neverthless what a jacobian or eigen value was, or how in the world space-state-modeling worked. I spent the first have of my internship going through lectures on MIT Opencourseware and watching Youtube videos in order to understand what I was supposed to do for my project. Since my mentor was assigned a heavy workload, she was unable to teach me everything I needed to know. This project taught the power of Google, and how to teach myself difficult concepts.

A basic over of Panel Code works:

The Panel Code is a code that projects objects into hypersonic flow, which is any flow above Mach 5. Newton's model for fluid flow states that in hypersonic flow, horizontal flow lines impact the shock wave created by the motion of the body, causing the flow to move tangential to the body. This can be used to estimate the pressure distribution over the body, estimating the force per unit area to be: N⁄A = p∞2*sin2(θ) This equation can be used to estimate the aerodynamic coefficients acting on a body.
